1. Add Language in Source Code

This master used to add new languages in the application. If user wanted to add new language in the application, then user needs to convert language file directory from the laravel application source code.
Steps to convert Language: Example if you wanted to add Xyz Language in the application.

  • Copy source_code/lang/en/*, where en is the English language directory.
  • Create new directory xy in same directory, source_code/lang/xy/*.
    Here, you need to copy en directory files in xy directory.
  • From the xy directory you need to open each file and need to convert English words in your Xyz language and save it.
