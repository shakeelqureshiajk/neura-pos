Important Notice

Please ensure you create a backup of your database and source code before proceeding with the update.

Database Backup Guide
1 Log in to phpMyAdmin
2 Select your database from the left sidebar
3 Click on the Export tab at the top
4 Choose Custom export option
5 Set format to SQL
6 Click Go to download the backup
Watch Video Tutorial
Update Process

Follow these steps to update your application:

  1. Complete the installation steps
  2. Transfer images from old version to new version
Image Transfer Path
source_code/storage/app/public/images/

Copy these folders:

  • app-logo
  • avatar
  • company
  • fevicon
  • items
  • services
  • signature

Final Step: Access your application URL and login

Examples:
  • https://example.com
  • https://subdomain.example.com
  • http://localhost/yourapp/
